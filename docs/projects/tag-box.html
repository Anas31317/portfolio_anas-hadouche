Web Industrie TPM · Lean En cours

Digitalisation Tag Box

Développement du système TPM Silver Level — Lear Corporation, Meknès

Entreprise Lear Corporation · Meknès, Maroc
Période Mai – Juillet 2026
Rôle Stage assistant ingénieur
Problème

Un système papier à digitaliser

Dans la zone de coupe de Lear Corporation à Meknès, le signalement des problèmes (pannes, non-conformités, incidents) se faisait entièrement sur papier via des "Tag Box". Ce système engendrait des pertes d'information, des délais de traitement et rendait impossible tout suivi en temps réel des indicateurs de performance (OEE, TRS).

Solution

Une application web full-stack

Développement d'une application web interne remplaçant intégralement le système papier. L'application gère les signalements, leur suivi, leur résolution et l'analyse des données via un tableau de bord temps réel.

3 rôles utilisateurs

Opérateur · Technicien · Chef d'équipe — chaque rôle dispose de droits et vues spécifiques.

Interface bilingue FR/AR + RTL

Support complet de l'arabe avec affichage RTL pour les opérateurs marocains.

Tableau de bord temps réel

Visualisation live des problèmes ouverts, temps de résolution, taux de récurrence et indicateurs TPM.

Stack technique
Backend
Python Flask MySQL SQLAlchemy
Frontend
HTML / CSS / JS Jinja2 Support RTL
Méthodes
